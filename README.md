# Bluestock Mutual Fund Analytics

## Project Overview

Analytics platform to study mutual fund performance,
risk metrics and investor behavior.

---

## Features

- Data cleaning pipeline
- Performance analytics
- Risk analysis
- Power BI dashboard


## Installation

Clone repository:

git clone <repo-url>


Install dependencies:

pip install -r requirements.txt


## Running ETL

python scripts/run_pipeline.py


## Dashboard

Open:

dashboard/bluestock_mf_dashboard.pbix


## Dataset Description

|File|Description|
|-|-|
|nav_history|Daily NAV|
|transactions|Investor activity|
|performance|Fund metrics|
